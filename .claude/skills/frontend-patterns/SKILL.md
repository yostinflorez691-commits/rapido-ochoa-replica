# Frontend Patterns

## Componentes React
- Functional components con arrow functions
- Props tipadas con interfaces (no types)
- Destructurar props en parámetros
- Export default al final
- Un componente por archivo

## Estilos
- Tailwind CSS mobile-first
- cn() para clases condicionales
- CSS variables para design tokens
- Framer Motion para animaciones

## TypeScript
- Strict mode siempre
- Nunca any (usar unknown si necesario)
- Interfaces sobre types
- Generics cuando aplique

## Accesibilidad
- Semantic HTML obligatorio
- Alt text en imágenes
- ARIA labels donde necesario
- Focus visible
- Contraste WCAG AA
